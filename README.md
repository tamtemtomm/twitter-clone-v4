## Getting Started
1. Clone the repo using this code <br/>
    <br/>
   `git clone https://github.com/tamtemtomm/twitter-clone-v4`
   <br></br>
3. Install the npm packages <br/>
   <br/>
   `cd twitter-clone-v4 && npm run build`
    <br></br>
4. Start the app <br/>
   <br/>
   `npm start` <br/>
